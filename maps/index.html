    Features in Map

    For map data, Highchats provides geometry data levels from World to US State County level (hc-key/fips/name). There is no Zip Code level data can be found on Highcharts Maps resources.

    Federal Information Processing Series (FIPS)

    FIPS Codes

    A FIPS code or Federal Information Processing Standard code, a unique numeric code that identifies geographic areas in the United States:

    • States: Two-digit codes
    • Counties: Five-digit codes, with the first two digits indicating the state
    • Other areas: alphabetically assigned codes for places, county subdivisions, consolidated cities, and American Indian, Alaska Native, and Native Hawaiian area.
    hc-key

    Unique hierarchical identifier for the area. Consistent across maps, and reflected in the naming convention of the maps.

    e.g. us, us-ma-027, gb-hi

    hc-a2

    Two letter abbreviation of area name. Usually same as postal code/ISO code, but not always. Not guaranteed to be unique within map.

    e.g. KV, PA, BB
